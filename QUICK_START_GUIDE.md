## üìû Common Questions

### Q: How do I add more blog posts?
A: Copy a `.post-card` div in the "Featured Posts" section and update the title, description, and link.

### Q: How do I change the products?
A: Edit the `.product-card` elements in the "Shop Products" section. Update title, price, rating, and affiliate link.

### Q: How do I add my own images?
A: Replace the placeholder URLs with paths to your images:
```html
<!-- Old -->
<img src="https://via.placeholder.com/..." alt="...">

<!-- New -->
<img src="./images/my-image.jpg" alt="...">
```

### Q: How do I make the newsletter form work?
A: Connect to email marketing service:
- Mailchimp: Get embed code, replace form
- ConvertKit: Use form embed
- Zapier: Automate form submissions

### Q: How do I add Google AdSense?
A: Add this code after `</head>` in `index.html`:
```html
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script>
```

### Q: How do I track affiliate sales?
A: Each affiliate program has its own tracking dashboard:
- Amazon: Go to affiliate account ‚Üí performance
- Other networks: Dashboard shows clicks and conversions

## üÜò Troubleshooting

### Website won't load
- Check file paths are correct
- Verify all files are in same folder
- Try a different browser

### Mobile version looks wrong
- Check viewport meta tag
- Ensure CSS media queries apply
- Test in DevTools mobile view

### Forms not working
- Check JavaScript console for errors
- Verify form input names
- Test email validation

### Links not working
- Check URL is correct
- Ensure file exists at path
- Use relative paths: `./file.html`
